Overview

CodeVision NZ is an AI-powered personal project designed to simplify information retrieval from the New Zealand Building Code documentation. Leveraging open-source technologies, it efficiently extracts, indexes, and retrieves textual and visual data from PDF documents.

Workflow

  • Data Ingestion & Indexing:
    • Locate and load PDFs matching a prefix/suffix.
    • Extract text chunks and embedded images simultaneously.
    • Image Extraction:
      • Store raw image data from each PDF.
      • Run a vision model to generate metadata and embeddings.
      • Save image embeddings into the Chroma collection.
    • Text Extraction:
      • Split raw text into manageable chunks.
      • Generate embeddings for each text chunk.
      • Save text embeddings into the Chroma collection.
  • User Query & Retrieval:
    • Receive the user‚Äôs question and load conversation context.
    • Retrieval Nodes:
      • Text node retrieves relevant text embeddings from Chroma.
      • Image node retrieves matching image embeddings from Chroma.
      • Merge text and image results into a single context payload.
    • Use an LLM to generate a final answer from the combined context.
    • Return the generated answer to the user.

    • Tools and Methods
