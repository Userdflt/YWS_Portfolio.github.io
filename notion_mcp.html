FastMCP Server

FastMCP Server enables real-time interaction with Notion using AI agents. It provides:

  • Tool Registration:
    • Content: append_content, create_table
    • Page: get_page_text, update_page_title, create_subpage
    • User: list_users, retrieve_user, get_me
    • Search: search_notion, retrieve_page_property
    • Media: text_to_speech
  • Prompt Endpoints:
    • default_prompt ‚Äì general tasks
    • structured_notes_prompt ‚Äì multi-section notes
  • Transport: Uses SSE at /sse for low-latency, push-style communication.

LangChain Agent

The LangChain Agent serves as the orchestration layer that powers intelligent behavior over Notion content by leveraging tools and a language model. It includes the following key components:

  • Tool Loading: Automatically discovers and injects all available MCP tools into the LangChain runtime, allowing dynamic tool use without manual configuration.
  • LLM: Defaults to gpt‚Äë4o via ChatOpenAI, with easy swapping to any Ollama or OpenAI-compatible model using a single flag.
  • Helpers: Provides built-in wrappers like summarize for condensing long text and speak for converting content into MP3s.
